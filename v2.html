/ LLM

ما تعرفه فعلًا — ChatGPT، Claude، Gemini.

// large language model

LLM هو نموذجُ ذكاءٍ اصطناعيٍّ مُدرَّبٌ على نصوصٍ ضخمة، يَتنبَّأ بالكلمة التَّالية بناءً على ما سَبَق.

المنظور الأبسط: function تأخذ نصًّا، وتُرجع نصًّا.

PROMPT ◀
«ترجم 'hello world' للفرنسيَّة»
f(x)
LLM
stateless · text → text
OUTPUT ▶
«bonjour le monde»

بلا ذاكرة، بلا أدوات، بلا مُبادرة. التَّطبيقات الشَّهيرة هي واجهاتٌ فوق LLMs.

// two ceilings

خاصَّتان تُحدِّدان LLM.

01 / LIMITED KNOWLEDGE
معرفة محدودة.

مُدرَّب على بياناتٍ عامَّة. لا يعرف calendarك، ولا ملفَّاتك، ولا مستندات شركتك الدَّاخليَّة.

اسأله: «ما سعر سهم Aramco الآن؟» → لا يعرف.
اسأله: «من ردَّ على PR رقم 42؟» → لا يعرف.
// trained ≠ connected
02 / PASSIVE
سلبيّ.

ينتظر prompt، ثمَّ يُجيب. لا يُبادر، لا يتصرَّف من نفسه.

لن يُنبِّهك: «اجتماعك خلال 5 دقائق».
لن يقول: «وجدت bug، أصلحته».

/ Prompt Engineering

كيف تتحدَّث مع النَّموذج.

// don't

أربعةُ أنماطٍ تُفسِد النَّتيجة.

01 · VAGUE
غامض.
«اكتب لي شيئًا عن AI»
لا يَعرف الجمهور، النَّبرة، الطُّول. تأتي إجابةٌ عامَّةٌ لا تَخدمك.
02 · NO CONTEXT
بلا سياق.
«أصلِح هذا الكود» — بلا الكود.
يَفترض أنَّ النَّموذج يقرأ ذِهنك. يَخمِّن. يُجيب عن شيءٍ آخر.
03 · COMPOUND
أسئلةٌ في كومة.
«اشرح REST، وقارِن GraphQL، واكتب API بـPython»
3 مَهام في prompt واحد ← 3 إجاباتٍ سطحيَّة، لا واحدة عميقة.
04 · NO FORMAT
بلا تَنسيق.
«لخِّص لي هذا»
قد يأتي نقاطًا، أو فقرة، أو markdown. لا تَستطيع البناء فوق ما لا تَتنبَّأ به.
// do

أربعةُ قواعدٍ تَرفع الجودة.

01 · BE SPECIFIC
كن محدَّدًا.
«اكتب 3 جُمل عن AI لمراهق، نبرةٌ حماسيَّة»
جمهور + نبرة + طُول = مساحةُ جوابٍ أضيق وأدقّ.
02 · GIVE CONTEXT
أعطِ السِّياق.
«هذا كود Python فيه TypeError بالسَّطر 12: [الكود]. أصلِحه واشرح.»
كلُّ ما يَعرفه عن المشكلة = كلُّ ما تَكتبه له.
03 · SET A ROLE
أعطِه دورًا.
«أنت محرِّرٌ صحفيٌّ متمرِّس. اقصُر كلَّ جُملةٍ تَتجاوز 20 كلمة.»
«أنت X» يَنقل أسلوبه لمنطقة المُحترفين في X.
04 · SPECIFY FORMAT
حدِّد الشَّكل.
«أعطِني الإجابة JSON بالحقول: title, summary, tags[3]»
تَنسيقٌ محدَّد = إخراجٌ يُمكن البناء فوقه برمجيًّا.

// the mechanism

الحَلقة.

ReAct · REASON + ACT

LLM يُفكِّر: ما الأداة الصَّحيحة الآن؟ يُنفِّذ: يستدعيها. يُراقب النَّتيجة. يُقرِّر: انتهيت، أم أُكرِّر؟

// a real trace · "ابحث عن تذاكر الرِّياض→جدَّة"
Reason: أحتاج flight search tool.
Act: flights.search(RUH→JED, 2026-05-10)
Observe: 14 نتيجة، أرخصها 420 ر.س.
Reason: المُستخدم طلب الأرخص → انتهيت.
while (!done) {
  thought  = llm.reason(ctx);
  result   = tool.run(thought);
  ctx.push(result);
  done     = llm.check(ctx);
}
