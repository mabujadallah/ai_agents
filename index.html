01
/ ما هو؟

عَرِّف الشيء. افصله عمَّا يُشبهه. واكتشف الجزء الأهمَّ.

// definitions

LLM · Chatbot · Agent

01 / BASE MODEL
LLM

دالَّة. مُدخَلٌ نصِّيٌّ يدخل، نصٌّ ناتجٌ يخرج. لا Memory بين الاستدعاءات، ولا أيدٍ.

verb → answers
02 / WRAPPER
Chatbot

LLM + حالة محادثة. يتذكَّر الجلسة، لكن بلا يدٍ تمتدُّ إلى العالم.

verb → converses
03 / AGENT
Agent

LLM مع Tools وLoop. يُفكِّر، يعمل في العالم، يُراقب، ويستمرُّ حتى تكتمل المُهمَّة.

verb → thinks, acts, loops
// focus here
// the mechanic

The Loop.

كلُّ Agent ببساطة: Think، Act بأداة، Observe النتيجة، ويُقرِّر إن كان سيُكرِّر. 

while (!done) {
  thought  = llm.plan(ctx);
  result   = tool.run(thought);
  ctx.push(result);
  done     = llm.check(ctx);
}
// definition
Agent =LLM +Tools +Memory +Loop
أربعة أجزاء. أزِل أيًّا منها تجد شيئًا آخر.
// anatomy

ممَّ يتكوَّن Agent.

01
LLM
// THE BRAIN

يُفكِّر في الخطوة التالية. يُخطِّط، يُفصِّل، ويحكم.

02
Tools
// THE HANDS

تمتدُّ إلى العالم. ملفَّات، واجهات، أوامر نظام، متصفِّحات، قواعد بيانات.

03
Memory
// THE NOTEBOOK

تُمسِك بالخيط. محادثة، وثائق مُسترجعة، ونتائج وسيطة.

04
Loop
// THE ORCHESTRATOR

يربطها. يُدير الدورة حتى تنتهي المُهمَّة — أو تنتهي الميزانية.

02
/ خمس خصائص

القُدرات الخمس التي تفصل الـAgent عن نافذة الـChatbot.

01
feature · 01 / 05

Tool Use.

لا يكتفي بالإجابة — يستدعي دوالَّ. يُعدِّل ملفَّات. يُشغِّل أوامر. يستدعي واجهات. يبحث في الويب.

// in the wild
Claude Code يُعدِّل الملفَّات، ويُشغِّل الاختبارات، ويفتح طلبات الدمج.
02
feature · 02 / 05

Reasoning & Planning.

فكِّك الهدف إلى خطوات، واحتفظ بالخُطَّة في ذهنك عبر جولاتٍ كثيرة دون أن تفقد الخيط.

// in the wild
الـDesign Agent في Google Stitch يُفكِّر عبر تاريخ المشروع كاملًا، لا آخر مُطالبة فحسب.
03
feature · 03 / 05

The Loop.

Autonomy. كلُّ خطوةٍ مبنيَّة على نتيجة سابقتها، وتستمرُّ دون إنسانٍ بين الجولات.

// in the wild
MManus يُشغِّل Browser Workflow تلقائيًّا لساعات دون إنسانٍ في الـLoop.
04
feature · 04 / 05

Memory & Context.

اقرأ المحيط. الـAgent يسحب حالة المشروع لتأتي مُخرجاته مُتَّفقة مع طريقة عمل الفريق.

// in the wild
Figma AI Agents يسحبون مكتبة المُكوِّنات والمُتغيِّرات والرُّموز كاملةً، لتأتي النتيجة مُتوافقة مع نظام تصميم الفريق.
05
feature · 05 / 05

Self-Correction.

يُلاحظ الفشل. يقرأ الخطأ. يُحاول مرَّةً أُخرى. هذا ما يُحوِّل العرض التجريبيَّ إلى مُنتَج.

// in the wild
Cursor يُعيد المحاولة بعد فشل الاختبارات. Claude Code يُصلِح أخطاء الإنتاج بقراءة السجلَّات والتكرار.
