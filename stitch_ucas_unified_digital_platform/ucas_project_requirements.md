# UCAS Digital Platform PRD

## 1. Project Overview
Design a modern, clean, and highly usable UI/UX for the University College of Applied Sciences (UCAS) digital platform, catering to students, faculty, and administrators.

## 2. Visual Identity
- **Style:** Minimalist, professional academic.
- **Colors:** Deep Blue (#1E3A8A), Teal (#06B6D4), Light Gray.
- **Typography:** Poppins (Headings), Roboto (Body).
- **UI Elements:** 8-16px rounded corners, soft shadows, high accessibility contrast.

## 3. Core Features & Screens
### Mobile App (Student Focus)
1. **Home Dashboard:** Personalized greeting, quick actions (Schedule, Courses, Grades), announcements, events.
2. **Courses Page:** Card-based layout with progress bars, materials access, and filters.
3. **Schedule:** Weekly color-coded calendar.
4. **Assignments & Exams:** List view with deadlines, status indicators, and upload functionality.
5. **Grades Dashboard:** Visual charts and GPA summary.
6. **Notifications:** Categorized list (Academic, Administrative, Events).
7. **Profile:** Student info, academic details, and language settings (English/Arabic).

## 4. UX Principles
- **Fast Navigation:** Bottom navigation bar.
- **Localization:** Full LTR/RTL support for English and Arabic.
- **Consistency:** Unified component system.
- **Accessibility:** Readable font sizes and high contrast.

## 5. Components
- Primary/Secondary/Disabled Buttons
- Input Fields
- Course & Event Cards
- Bottom Navigation Bar
- Sidebar (for Desktop views)
- Modals & Toast Notifications
